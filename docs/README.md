# @authumn/angular

## Installation

To install this library, run:

```bash
$ npm install @authumn/angular --save
```

App must setup ngrx module.





## Development

To generate all `*.js`, `*.d.ts` and `*.metadata.json` files:

```bash
$ npm run build
```

To lint all `*.ts` files:

```bash
$ npm run lint
```
